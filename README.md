Fake News Detection using Machine Learning

A professional Machine Learning project that detects whether a news article is Real or Fake using Natural Language Processing (NLP) techniques and Logistic Regression.

🚀 Project Overview

This project analyzes news articles and predicts their authenticity using:

TF-IDF Vectorization
Logistic Regression
NLP techniques
Scikit-learn

The model was trained on real and fake news datasets and achieved an accuracy of 98%.
